## 文案語言風格分析

### 英文文案（約 55%）
**用途**：技術案例、工具開箱、國際活動  
**特色**：
- 技術描述清晰，有工具清單（bullet point 格式）
- 口語化，不官腔（"these are no joke"、"my new favorite 😅"）
- 誠實分享失敗（"Yes, I failed on this one"）→ 建立可信度
- 結尾 hashtag 10-15 個（英文＋中文混搭）

**範例模式**：
```
[挑戰/問題描述] → [解決方式/工具介紹] → [結果/學習] → hashtags
```

### 中文文案（約 35%）
**用途**：比賽成果、公益活動、情感故事、在地業務  
**特色**：
- 換行斷句，節奏感強（每句單獨一行）
- 口語直接，有台灣話語感
- 情感敘事強（老車故事那篇爆文風格）
- 含實際資訊（地址、電話、報價）

**範例模式（情感類）**：
```
[情境敘述，短句換行]
「引述客戶的話」
[行動/結果]
[感謝/呼籲]
```

### 中英混合（約 10%）
**用途**：培訓推廣、活動宣傳  
- 標題用中文，說明用英文  
- 同時觸及台灣粉絲和國際 PDR 社群
